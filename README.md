# НЦДПИиР — виртуальный тур 360°

Интерактивный панорамный тур по залам Национального центра декоративно-прикладного искусства и ремёсел.

**Демо:** https://nyrokume.github.io/ncdpir-virtual-tour-360/

## Локальный запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Сайт публикуется на GitHub Pages автоматически при push в ветку `main`.
